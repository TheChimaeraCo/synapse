# Meal Planner

Generated by Synapse Module Forge.

## Request
Create a meal planning module with the following features:

1. Recipe Management - save recipes with name, ingredients, instructions, tags, servings, nutrition info
2. Meal Planning - create weekly meal plans, assign recipes to specific days and meal types (breakfast, lunch, dinner, snack)
3. Shopping List Generation - auto-generate shopping lists from meal plans, aggregate ingredients intelligently by category
4. Recipe Search - search by name, tags, ingredients
5. Nutrition Tracking - store and retrieve nutrition info per recipe

The module should be conversational and easy to use without knowing function names.

## Module ID
`meal-planner`

## Tools
- `meal-planner.save_recipe`: Save or update a recipe with ingredients and instructions.
- `meal-planner.list_recipes`: List or search saved recipes by query or tag.
- `meal-planner.plan_week`: Create or replace a weekly meal plan using saved recipe ids.
- `meal-planner.shopping_list`: Generate an aggregated shopping list from a weekly meal plan.
- `meal-planner.nutrition_lookup`: Lookup nutrition facts for an ingredient/food via OpenFoodFacts.
